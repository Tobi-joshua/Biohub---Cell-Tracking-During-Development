# Data Notes ## Training data - `.zarr` image volumes - paired `.geff` graph annotations ## Test data - `.zarr` image volumes only ## Volume format - shape: `(T, Z, Y, X)` - one timepoint per chunk - physical voxel scale: - z = 1.625 µm/voxel - y = 0.40625 µm/voxel - x = 0.40625 µm/voxel ## Ground truth - nodes contain centroid coordinates - edges link cells across time - labels are sparse ## Submission graph - node rows for detections - edge rows for lineage links - node rows contain `node_id, t, z, y, x` - edge rows contain `source_id, target_id`
