# Competition Rules Summary

## Competition
Biohub - Cell Tracking During Development

## Core task
Track zebrafish cells through 3D space and time from fluorescence microscopy volumes.

## Submission format
- CSV file named `submission.csv`
- Must include both `node` and `edge` row types
- Must contain all datasets in the hidden test set
- Columns:
  `id,dataset,row_type,node_id,t,z,y,x,source_id,target_id`

## Notebook rules
- Submissions must be made through Kaggle Notebooks
- CPU notebook runtime limit: 12 hours
- GPU notebook runtime limit: 12 hours
- Internet access disabled in submission notebook
- External data is allowed if it is publicly available, equally accessible to all participants, and reasonably accessible at minimal cost

## Competition limits
- Maximum 5 submissions per day
- Team size limit: 5
- Team mergers allowed before the deadline
- Use one Kaggle account only

## Important dates
- Start: June 29, 2026
- Entry deadline: September 22, 2026
- Team merger deadline: September 22, 2026
- Final submission deadline: September 29, 2026

## Prize pool
- 1st: $18,000
- 2nd: $12,000
- 3rd: $8,000
- 4th: $6,000
- 5th: $6,000
- 6th: $5,000
- 7th: $5,000

## External data
External data may be used only if it is public, accessible to all participants, and properly cited in the notebook.
